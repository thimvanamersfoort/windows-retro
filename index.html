About us

About this project:

We (Jesse Dragt and Thim v. Amersfoort) made this website as a school project for computer science. The goal was to create a responsive website using client-side code only. We decided to give this assignment our own twist, and we created Windows Retro. A fully functioning, client-side only web-experience that takes the visitor back to a simpler era. The design is sleek and simple, and the various "Applications" that we needed to develop gave us an opportunity to use new and unfamiliar plugins and technologies.

The technologies we used to make this project possible are not entirely Vanilla JavaScript. We used some JS plugins / libraries for functions like design, responsiveness and ease of coding. These plugins (and other dependencies) we used are listed below. We also made sure to keep our HTML / CSS layout as general and accessible as possible. We included a basic documentation of our code in the File Explorer program, under the folder Documents.

If you're interested in building your own applications / infrastructure with this design, you can download the source-code using the link below (direct download). The source code is free, so please refrain from (re)selling versions of it. Also, credits to our work would be appreciated.

Plugins and dependencies:

  1. jQuery: A lightweight JS library that simplifies and shortens JS code, which comes with many different prebuilt functions. Used regularly in the entire project.
  2. jQuery UI: An extension of jQuery that is used for more complicated UI code. Used in the project to implement drag&drop.
  3. jQuery Touch Punch: A small extension that overrides normal jQuery UI click-events with their corresponding touch-events, to ensure compatibility on mobile devices.
  4. TypeIt: An animation library we used for the reboot- sequence. It simplifies the code to create a typewriter-like effect / animation.
  5. Windows 95 UI Kit: A CSS template we used for the basic design of buttons, cards, and icons.
  6. Windows 98 Icons: A free Windows 98 icon repository, created by Alex Meub. This is where almost all of the icons we used come from.
